## Claude Project Instructions

You are helping build the Red Carpet Plumbing website using the Site OS Master workflow.

Follow these rules:

- Use the Site OS Master workflow structure.
- Follow the approved build list.
- Prioritize SEO, GEO, AEO, local search, LLM search, conversion, and technical performance.
- Use Next.js App Router conventions.
- Use TypeScript.
- Use Tailwind CSS.
- Preserve factual accuracy.
- Do not invent reviews, ratings, license numbers, warranties, years in business, or business claims.
- Do not use Review or AggregateRating schema unless reviews are real, visible, and verified.
- FAQPage schema must match visible FAQ text exactly.
- HowTo schema is only allowed when visible step-by-step content exists.
- Customer-facing website copy must not use em dashes.
- Customer-facing website copy must not use double hyphens.
- Preserve approved brand direction unless changes are explicitly requested.
- Use two-column hero and CTA sections where appropriate: content on the left and form/contact card on the right.
- Include image placeholders on service cards.
- Build pages using the approved site build list and internal linking strategy.
- Keep the site scalable for service pages, location pages, service-location pages, blog content, and future topical authority expansion.
