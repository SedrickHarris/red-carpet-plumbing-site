# Las Vegas Plumbing Website Build List & Internal Linking Strategy

## Project Goal

Build a high-performing Las Vegas plumbing website optimized for:

- SEO
- GEO targeting
- AEO
- Local intent SERP visibility
- Google Search Console
- Bing Webmaster Tools
- Apple Business Connect / Apple Maps
- Rich snippets
- Featured snippets
- People Also Ask
- Google Business Profile support
- Knowledge panel support
- Perplexity visibility
- ChatGPT and LLM search visibility
- Topical authority
- Customer engagement
- Customer conversion

The site should be fast, crawlable, locally relevant, service-specific, schema-ready, and structured to support both human users and AI/search retrieval systems. The build should include city/service pages, FAQ and Service schema, GBP support, citation consistency, review workflows, and measurement through GSC, Bing Webmaster, analytics, and local search tools.

---

# 1. Core Build Principles

## 1.1 Content Structure Rule

Every page should follow this structure:

Search intent → local relevance → direct answer → service details → trust proof → FAQs → CTA → schema

## 1.2 Conversion Rule

Every page should include clear conversion paths:

- Click-to-call CTA
- Request service CTA
- Short contact form
- Emergency CTA where relevant
- Trust signals near conversion points

## 1.3 Local SEO Rule

Every location and service-location page should clearly answer:

- What service is offered?
- Where is it offered?
- Who is it for?
- What problem does it solve?
- Why should the customer trust the company?
- What should the customer do next?

## 1.4 Schema Rule

Use structured data only when it matches visible page content.

Recommended schema types:

- LocalBusiness / Plumber
- Organization
- WebSite
- WebPage
- Service
- FAQPage
- BreadcrumbList
- Article / BlogPosting
- HowTo, only when visible step-by-step content exists
- Review / AggregateRating, only when reviews are real, visible, and verified
